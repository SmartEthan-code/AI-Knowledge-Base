# CNN和计算机视觉学习指南

## 1. 基础概念

### 卷积神经网络(CNN)
- **卷积层**: 提取图像特征
- **池化层**: 降维和特征聚合
- **全连接层**: 分类决策

### 计算机视觉(CV)
图像处理和分析的核心任务：
- 图像分类
- 目标检测
- 语义分割
- 实例分割

## 2. 经典CNN架构

| 架构 | 年份 | 用途 |
|------|------|------|
| LeNet | 1998 | 手写数字识别 |
| AlexNet | 2012 | ImageNet竞赛 |
| VGG | 2014 | 深层网络设计 |
| ResNet | 2015 | 残差学习 |

## 3. 学习路径

```
基础数学 → CNN原理 → 实现框架 → 经典模型 → 应用项目
```

## 4. 推荐框架

- **PyTorch**: 灵活易用
- **TensorFlow**: 生产部署
- **OpenCV**: 图像处理工具
