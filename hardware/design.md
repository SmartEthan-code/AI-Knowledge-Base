# 硬件设计指导手册

## 1. 概述
简要说明设计目的、目标产品、预期使用环境与关键性能指标（KPI）。

## 2. 需求与约束
- 功能需求：列出必须实现的功能点。  
- 性能需求：功耗、响应时间、带宽、精度等。  
- 物理约束：尺寸、重量、环境温度、振动等级。  
- 合规与认证：EMC、CE、FCC、UL、RoHS 等。  
- 成本与时间表：目标单价、开发里程碑。

## 3. 系统架构
- 模块划分：电源、处理单元、传感/驱动、通信、I/O。  
- 接口定义：电气接口（电压、电流、协议）、机械接口、软件 API。  
- 冗余与故障处理策略。

## 4. 关键组件选择原则
- 可靠性优先：选择长期可得、寿命和温漂满足需求的器件。  
- 可获得性：供应链风险、替代料号策略。  
- 成本与性能平衡。  
- 标准化：优先使用标准封装和接口。

## 5. 原理图设计要点
- 电源完整性：滤波、去耦、电源轨分割、保护（TVS、熔断器）。  
- 信号完整性：阻抗匹配、差分对、终端、走线长度控制。  
- 接地策略：星形地、数字/模拟分割、地平面。  
- 器件封装与符号一致性、清晰标注参考设计ator。

## 6. PCB 布局与布线
- 分层规划：电源层、地层、信号层布局。  
- 高速信号：最短路径、避免跨层跳线、差分间距控制。  
- 区域划分：模拟/数字/射频/电源隔离。  
- 散热：热铜通、散热片、热仿真评估。  
- 制造公差：最小走线/间距、过孔尺寸、丝印规则。

## 7. 机械与连接器设计
- 机电界面：固定孔、卡扣、密封措施。  
- 连接器选型：极数、编码、上/下/侧向可插性。  
- 机箱接地与屏蔽策略。

## 8. 固件与测试点
- 提供调试接口（JTAG、SWD、UART）。  
- 设计足够的测试点/测试夹具アクセス点以便生产测试。  
- Bootloader 与安全引导策略。

## 9. 测试验证计划（V&V）
- 单元测试：器件级功能验证。  
- 系统测试：功能、性能、边界条件。  
- 环境测试：温度、湿度、振动、冲击。  
- EMC/安全测试：辐射、传导、绝缘、漏电流。  
- 老化与可靠性：加速寿命测试、MTBF 估算。

## 10. 量产准备
- 生产文件：BOM（含替代料号）、装配图、料表、程序烧录文件。  
- 测试程序：ICT、FCT、功能测试流程与判定标准。  
- 质量控制：进料检验、首件检验、失效分析流程。  
- 供应链管理：长期采购计划、最小订货量、JIT 考量。

## 11. 文档与版本控制
- 使用版本控制管理原理图、PCB、BOM 与固件（如 Git）。  
- 维护变更日志（ECN/ECM）、设计审核记录与批准签名。  
- 生成维护手册、服务指南与保修条款。

## 12. 安全与合规注意事项
- 电气安全：爬电/爬痕距离、隔离、绝缘等级。  
- 电磁兼容：滤波、屏蔽、接地处理。  
- 环保：材料与工艺符合 RoHS/REACH 要求。  
- 数据安全：若含通信/存储，考虑加密与认证机制。

## 13. 常用检查清单（示例）
- 电源：去耦电容、稳压、过流保护检查。  
- 接地：地平面连续性、焊接点完整性。  
- 信号：差分对长度匹配、终端电阻。  
- 机械：安装孔位置与间隙。  
- 文档：BOM 完整性、替代料号记录、制造文件。

## 14. 参考流程（简化）
1. 需求定义 -> 2. 概念设计 -> 3. 原理图与器件选型 -> 4. PCB 布局 -> 5. 样机制作 -> 6. 测试与修正 -> 7. 认证 -> 8. 试产 -> 9. 量产。

## 15. 结语
保持模块化设计、早期关注可制造性与可测试性、并在关键节点进行设计评审以降低风险。

# 硬件开发日记 1
- 导热垫不要含硫，氯，包括卤素等容易腐蚀磁珠（铁氧体）内部电路，时间长了之后导致器件开路
- 硫化发黑，防硫化电阻，三防漆
- 需要涂三防漆的产品要考虑防硫化，有机硅类三防漆含硫，硅会吸附空气中的硫，可使用有机硅+树脂打底
- 产品或物料的存储周期不能太长
- 导热垫渗油率考量，一般不能太高
- 导热垫不能太大，尽量避免直接贴在有大量器件的上方

# 硬件开发日记 2
- 氨极易腐蚀铜
- 泡棉生产中如果使用化学发泡剂制程，有一种使用甲酰胺工艺的，如果挥发程度不够就遗留下来，在空气湿度很大的情况下遗留下来，造成铜腐蚀

# 硬件开发日记 3
- 化学品相互反应，特别是挥发物需要特别注意
- SMT化学品兼容性实验，年检，两两兼容，锡膏和锡丝，助焊剂，清洗剂，全部混合（辅助物料混合使用兼容性评估报告，代工厂需要做，老化实验）
- 盐雾试验（器件内部实验，不是外观）元件选型，本体是否抗腐蚀
- 电化学迁移，负极枝状结晶
- 助焊剂含酸，助焊剂的主要目的就是去氧化，所以效果越强，其可靠性就越差，如果清洗不够，容易出问题
- 抑制电化学迁移，选择吸附湿性低的基板材料，保持干燥
- PCB内部铜迁移，一般是钻孔出问题

# 硬件开发日记 4
- via尽量不要并排设计，不要设计的太过于“整齐，漂亮”
- 错位和分开设计，保证距离
- 玻纤布选择编织布，开纤布
